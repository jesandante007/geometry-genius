Question

  1. What is dom? What is the purpose of dom?

    2. Answer: DOM means Document Object Model. Through DOM we can manipulate web page. DOM is an object tree of a html document.It's an programming interface for web documents. By this interface using JavaScript we can manipulate, change element, add or remove attributes, change style & so many things.

  2. How to select elements from dom?

    3. Answer: We can select dom elements in many ways. There are so many methods and tricks to grab elements or nodes from dom. We use then as per our needs. Some methods are given:
    document.getElementById()
    getElementsByTagName()
    getElementsByClassName()
    querySelector()
    querySelectorAll

  3. What is event bubble?

    4. Answer: Event bubble is a concept of DOM. Let's say there are three event inside one another. So if the child event happens then it's parent event and grand parent event automatically happen. If parent event happens it's parent will automatically happen. This is like bubble in the deep water and it's raising up. So the bubbling concept is similar to that, event will always raising up to it's ancestors

  4. What is callback function?

    5. Answer: Callback function is a function which is passed as a parameter of another function's. When main function's work done then it execute callback function. Usually callback function uses when synchronous execution needed. Because JavaScript is asynchronous, means doesn't wait for a function to to execute it's next but invoked immediately. So where synchronous behavior need use callback function
